# Raspberry Pi 5 Platform Assessment

**Status:** Planned  
**Target:** Raspberry Pi 5, 16 GB

## Question

What operating, resource, storage, and thermal constraints must the preview architecture respect on the target device?

## Evaluation Areas

- Supported 64-bit base operating system and maintenance lifecycle
- CPU, memory, storage, and thermal behavior
- Required cooling, power supply, and storage hardware
- ARM64 availability of candidate runtime dependencies
- Concurrent Pi-hole, platform, and AI workloads
- Boot, service supervision, and power-loss recovery
- Expected idle and active power consumption

## Required Evidence

- Exact hardware and OS versions
- Repeatable measurement commands
- Idle and representative-workload measurements
- Sustained-load temperature and throttling results
- Storage footprint and write behavior
- Limitations that affect support or installation guidance

## Decision Informed

Raspberry Pi deployment RFC and preview hardware requirements.
