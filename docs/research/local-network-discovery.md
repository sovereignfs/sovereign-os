# Local Network Discovery

**Status:** Planned

## Question

How should a user reliably find the Phase 01 Pi-hole appliance from common phones and computers on a home network?

## Alternatives

- Direct IP address as the primary method
- mDNS hostname
- Router-provided local DNS
- Pi-hole local DNS integration
- A combination with documented fallback order

## Evaluation Criteria

- Reliability across common clients and network layouts
- First-run usability
- Conflicts and hostname changes
- IPv4 and IPv6 behavior
- Security implications
- Dependency on Pi-hole availability
- Support and troubleshooting burden

## Decision Informed

Local networking and hostname RFC.
